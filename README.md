# archonit_demo

Test task demo project.

INITIAL REQUIREMENTS:
1. Implement a screen from figma adhering to all parameters (size, color, font). Each item has a 
    colored Container, there should be an option for each subsequent Container 
    in the ListView, to generate a background color from 16777216 colors using RGB.
    DESIGN: https://www.figma.com/file/ac3bb7tiNhB2bI9vETQxmA/Untitled?type=design&node-id=0%3A1&mode=design&t=sSEKyR3QE9hgA36M-1

2. Data from the REST Api, which is listed below. Implement pagination when scrolling down, 
    display the next 15 elements.
    REST API: https://docs.coincap.io/#ee0c0be6-513f-4466-bbb0-2016add462e9
    HTTP package: https://pub.dev/packages/dio

3. Submit to GitHub and send a link to the repo
   - compliance of the completed task with the specified requirements
   - code cleanliness
   - correct operation of the program
   - compliance with the rules for writing code.


To run the project, you should:
1. register on the resource by link REST API.
2. get an Api Key. 
3. create a file .env in root of your project.
4. put your Api Key to this file by key API_TOKEN=
